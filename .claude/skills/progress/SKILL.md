# 学习进度查看

读取 `progress/tracker.md` 文件，按以下格式展示：

1. **总体进度**：当前阶段、完成度、学习天数、会话次数
2. **模块状态**：列出各模块学习状态，重点标注 🔄 进行中和 ❓ 待学习的模块
3. **遗留问题**：列出待解决问题
4. **最近会话**：最近 3 次会话的主题和日期（从学习笔记索引表读取）
5. **建议**：根据当前进度建议下一步学习方向

使用简体中文输出，简洁明了。
